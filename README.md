### Installing

* Install all needed packages
```
npm i
``` 
* Install ts-node to run typescript-based code
```
npm install -g ts-node
```
* Install MongoDB for VS Code extension

### Executing program

* Run the project
```
npm start
```
* Stop running
```
^c
```
```
control + c
```
